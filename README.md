# Forecasting Penjualan Retail

## Deskripsi

Project ini dibuat sebagai tugas UAS Data Analytics.

Dataset yang digunakan adalah Customer Shopping Dataset dari Kaggle.

## Metode

- Exponential Smoothing
- ARIMA (1,0,0)

## Tools

- Python
- Pandas
- Streamlit
- Statsmodels

## Cara Menjalankan

Install library

```bash
pip install -r requirements.txt
```

Jalankan aplikasi

```bash
streamlit run app.py
```

## Hasil

Model terbaik adalah **ARIMA (1,0,0)**

- MAE : 202.829
- RMSE : 244.902
- MAPE : 2.19%

https://www.kaggle.com/datasets/mehmettahiraslan/customer-shopping-dataset
